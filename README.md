# micro-front

小規模なブログ機能メインのフロントページ

## 設計について

本プロジェクトの初期設計書は下記の通り。

## 推奨参照順
1. `docs/design/1.site-design.md`
2. `docs/design/2-1.detail-common.md`
3. `docs/design/2-2.detail-admin.md`
4. `docs/design/2-3.detail-public.md`

## ファイル概要

### `docs/design/1.site-design.md`
- 基本設計書
- 画面要件、URL要件、データ設計、全体方針を記載する

### `docs/design/2-1.detail-common.md`
- 共通詳細設計書
- 技術構成、DB設計、API共通方針、バリデーション、インデックス設計を記載する

### `docs/design/2-2.detail-admin.md`
- 管理ページ詳細設計書
- 管理API、管理ページレイアウト、管理画面用コンポーネント設計を記載する

### `docs/design/2-3.detail-public.md`
- 公開ページ詳細設計書
- 公開ページレイアウト、公開コンテンツ、静的HTML出力後の公開仕様を記載する
